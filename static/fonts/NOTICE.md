# fonts

`iAWriterDuoS-*.woff2` are **iA Writer Duo** (formerly "Duospace"), by iA Inc.
Released under the SIL Open Font License. Source:
https://github.com/iaolo/iA-Fonts (`iA Writer Duo/Webfonts/`).

`JetBrainsMono-*.woff2` are **JetBrains Mono**, by JetBrains s.r.o. Released under
the SIL Open Font License. Used for code blocks and gists. Source:
https://github.com/JetBrains/JetBrainsMono (`fonts/webfonts/`).

these are self-hosted to keep the site free of webfont-CDN dependencies. to swap
the body font, replace these woff2 files and update the `@font-face` blocks +
`--font-body` stack in `themes/botufte/assets/css/_base.scss`.
